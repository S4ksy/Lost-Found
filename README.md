# Campus Lost & Found System

A comprehensive web-based system for managing lost and found items on campus.

## Features

- **Report Lost Items**: Submit detailed information about lost items
- **Report Found Items**: Register found items with photos and descriptions
- **Item Catalog**: Browse and search through found items
- **Claim System**: Submit claims for found items with proof of ownership
- **Admin Management**: Review and approve/reject claims
- **User & Admin Views**: Toggle between different user roles
- **Responsive Design**: Works on desktop and mobile devices

## Quick Start

1. **Download all files** to a folder named `lost-and-found-system`
2. **Open `index.html`** in a web browser
3. **Start using the system** immediately!
